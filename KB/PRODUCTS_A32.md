# A32 – Scalable Immersion Cooling Tank
适用对象：AC40 / DC45 / 独立边缘部署的浸没式液冷算力单元
版本：V1.1（2026-03-29 统一结构版）

---

## 1. 产品定位

A32 是 32RU OU-compatible 浸没式液冷单机柜，单柜 IT 负载 45–50kW，采用 2N CDU 冗余设计，面向边缘计算与高密度 GPU 部署。

可单独部署，也可 8 台组合成 AC40 集装箱。

---

## 2. 核心参数

| 项目 | 参数 |
|------|------|
| IT 容量 | **45–50kW**（最大值 50kW） |
| Rack Units | 32RU |
| 冷却类型 | 单相浸没式液冷 |
| CDU 配置 | 2 × 50kW（**2N 冗余**） |
| 服务器进口温度 | 32–35°C |
| 服务器出口温度 | 35–38°C |
| PUE 设计值 | **取决于环境温度**：<br>• 环境 <28°C（纯干冷）：~1.03–1.05<br>• 环境 >28°C（DX 辅助）：~1.08–1.12 |
| 兼容标准 | OU / 29OU（OCP） |

---

## 3. IT负载 vs 整体电力负荷

| 项目 | 数值 | 说明 |
|------|------|------|
| **IT 负载** | 45–50kW | 服务器、GPU 实际消耗 |
| **整体电力负荷** | ~46–56kW | 取决于 PUE（PUE 约 1.03–1.12） |

---

## 4. 机柜详情

- 32RU 标准高度
- 内部集成 Dual CDU（1+1 冗余）
- OU-compatible / 29OU（OCP 兼容）
- 支持 OCP 认证服务器

---

## 5. 冷却架构

### 5.1 热力路径

```
IT Load → Dielectric Fluid → Heat Exchanger → CDU → **Hybrid Cooling System**
```

### 5.2 散热选型规则

| 环境条件 | 推荐散热方式 |
|----------|-------------|
| 环境温度 <28°C | 纯干冷器（允许） |
| 环境温度 ≥28°C | **必须**增加 DX 或热泵辅助（禁止纯干冷）|

> ⚠️ **重要规则：** A32 不允许使用纯干冷器作为唯一散热方式。环境 >28°C 地区必须配置 DX 辅助。

### 5.3 故障模式

| 故障 | 影响 | 风险等级 |
|------|------|----------|
| 单 CDU 故障 | 无影响（2N 冗余） | 低 |
| 双 CDU 故障 | 停机 | 高 |
| **Hybrid Cooling System** 失效 | 温度升高（DX 兜底） | 中 |
| Pump 失效 | 过热 | 高 |

---

## 6. 电力路径

```
INPUT → PDC → UPS（可选，外置） → A32 Tank → IT Servers
```

说明：
- A32 本体**不包含 UPS**（UPS 需外置或由 AC40/DC45 提供）
- 支持外部 PDC 接入
- 可对接 BESS 或发电机系统

---

## 7. 适用场景

| 场景 | 适用性 |
|------|--------|
| 高密度 GPU 训练 | ✅ 适合 |
| 边缘推理部署 | ✅ 适合 |
| 干燥气候区域（低温） | ✅ 适合（可纯干冷，PUE 低至 1.03）|
| 电价高区域 | ✅ 适合（低 PUE 优势）|
| 高湿无通风环境 | ❌ 禁止 |
| 无外部冷源场地 | ❌ 禁止 |
| 客户强制要求空气冷却 | ❌ 禁止 |

---

## 8. 扩展逻辑

| 扩展方式 | 说明 |
|----------|------|
| 独立部署 | 单 A32 可独立使用（需配置外部 PDC/UPS）|
| AC40 组合 | 8 台 A32 + AC40 集装箱框架 = 400kW IT |
| 集群扩展 | 多台 AC40 并联组成 MDC |

## 9. 与 AC40 / DC45 的关系

| 产品 | 定位 | IT 容量 | 冷却类型 |
|------|------|---------|----------|
| **A32** | 单柜单元 | 45–50kW | 浸没式 |
| **AC40** | 40ft 集装箱（8×A32） | 400kW | 浸没式 |
| **DC45** | 45ft 集装箱（8×DLC） | 1200kW | 直冷液冷 DLC |
| **MDC** | 模块化集群（AC40+DC45） | 可达 5MW+ | 混合 |
