# AC40 – All-In-One Immersion Container (40ft)
适用对象：MDC（Modular Datacenter Cluster）标准计算单元 IT Zone
版本：V1.1（2026-03-29 统一结构版）

---

## 1. 产品定位

AC40 是 40ft 容器规格的浸没式液冷集装箱，支持 8 × 50kW A32 浸没槽，适用于高性能 AI 算力部署。

标准 IT 容量为 400kW，支持与 DC45 混合构建 MDC 集群。

---

## 2. 核心参数

| 项目 | 参数 |
|------|------|
| IT 容量 | **400kW** |
| 浸没槽型号 | A32 |
| 浸没槽数量 | 8 × 50kW = 400kW |
| 风冷机柜 | 10kW × 1 |
| 服务器进口温度 | 32°C |
| 内阻 CDU | Dual CDU，1+1 完全冗余（内置于各 Tank） |
| UPS | EATON **9395XR-600**，4 个功率模块（每模块 150kW），总功率 **600kW**，发热量约 7.9kW |
| 电池 | 2 个 93LiG2 机柜，每柜 332kW，共提供 **10 分钟**后备时间 |
| 功率因数 | 0.9（UPS 输出侧） |

> **注：** 9395XR-600 中的"600"= 4 × 150kW 模块，与 DC45 使用的 9395XR-1500（10 模块）属同一产品系列，仅模块数量不同。

---

## 3. IT负载 vs 整体电力负荷

| 项目 | 数值 | 说明 |
|------|------|------|
| **IT 负载** | 400kW | 服务器、GPU 实际消耗 |
| **整体电力负荷** | ~440–480kW | 取决于 PUE（PUE 约 1.10–1.20，环境 >28°C 时 DX 辅助运行）|
| PUE 设计值 | **取决于环境温度**：<br>• 低温区（干冷器优先）：~1.08–1.10<br>• 高温区（DX 辅助）：~1.15–1.20 | 不写固定数字 |

---

## 4. 机柜详情

- 32RU / 29RU 两种配置
- 内阻 Dual CDU，1+1 完全冗余（各 Tank 内置）
- 双排靠集装箱墙壁摆放，中间预留运维通道
- 机柜底部 350–400mm 管路空间，配电区域无地板架高

---

## 5. 冷却架构

### 5.1 浸没冷却部分

| 项目 | 参数 |
|------|------|
| Tank 型号 | A32，50kW/柜 |
| Tank 数量 | 8 |
| 热交换器 | CDU（内置于 Tank），2N 冗余 |
| 散热方式 | **Hybrid Cooling System**（干冷器 + DX 一体化）或 **热泵** |
| 允许方式 | Hybrid Cooling System / 热泵 |
| **禁止** | 纯干冷器（无 DX）— 环境 >28°C 时无法满足散热需求 |

### 5.2 风冷部分

- 独立机柜 1 个（10kW）
- 必须配置独立风冷空调，不与浸没冷却共用

### 5.3 水路冗余

- 各 Tank 内置 CDU，无需独立 CDU
- 整体 2N 冗余设计

---

## 6. 电力路径

```
Grid / BESS → PDC → UPS (EATON 9395XR-600) → PDC → Tanks → PDU
```

说明：
- UPS 内置于 AC40，提供 5–10 分钟后备
- BESS 连接方式：Grid → BESS → AC40（Power Zone 负责）
- 也支持 Grid → ATS → Generator → AC40（视客户场地条件）

---

## 7. 冗余说明

> ⚠️ **重要区分：UPS 模块冗余 vs IT Zone 容器冗余**

| 层级 | 冗余描述 |
|------|---------|
| **UPS 模块** | 9395XR-600 内置 4 个功率模块，支持内部 N+1（单模块故障不影响运行）|
| **CDU** | 各 Tank 内置 Dual CDU，1+1 完全冗余 |
| **IT Zone（AC40 集装箱）** | **无内部冗余** — 单台 AC40 独立运行，不含内部 N+1/2N |
| **MDC 系统级** | 多台 AC40 并联 → 系统级冗余（N / N+1 / 2N 由集装箱数量决定）|

> AC40 是一台完整的集装箱设备，不存在"多增加几个服务器"来增加冗余的概念。
> 如需更高可靠性，通过增加 AC40 集装箱数量实现。

## 9. 扩展逻辑

- 可与 DC45 混合构建 MDC
- 支持横向扩展（增加 AC40 数量）
- 支持纵向扩展（增加单柜 GPU 密度，但需重新评估冷却）
